# Sprite Design Reference

## PixelLab Prompt Template

Use this base prompt structure for consistency:

```
[CHARACTER DESCRIPTION], pixel art, sprite, [THEME STYLE], simple shading, clean lines, retro game aesthetic
```

### Theme Style Modifiers

- **Fantasy**: medieval fantasy, warm colors, earthy tones, magical glow effects
- **Sci-Fi**: retro sci-fi, cool blues and purples, holographic accents, sleek design
- **Cyberpunk**: neon pink and cyan, dark background contrast, chrome accents, gritty urban
- **Cabin**: cozy rustic, warm oranges and browns, soft lighting, comfortable homey feel
- **Underwater**: deep ocean blues and teals, bioluminescent accents, flowing movement, bubble details
